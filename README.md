# graphql-magic

Welcome to graphql-magic!

## Usage

### Prerequisites

* Next.js
* TypeScript
* Knex.js
* Postgresql

### Setup

Dependencies:

```
npm i @smartive/graphql-magic
```

Setup:

```
npx gqm setup
```

Generate all the things:

```
npx gqm generate
```

Generate a migration:

```
npx gqm generate-migration
```
